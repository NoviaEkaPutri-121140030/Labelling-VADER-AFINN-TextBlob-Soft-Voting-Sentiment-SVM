# Labelling-VADER-AFINN-TextBlob-Soft-Voting-Sentiment-SVM
A sentiment analysis project whom compare the lexicon labelling(VADER, AFINN, TextBlob) with Soft Voting approch on Indonesian-language Online Travel Agent (OTA) review data translated into English using the DeepL Translator API and classified using SVM with various kernel variations.
